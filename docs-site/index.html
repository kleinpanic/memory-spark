Dynamic Reranker Gate

The key insight: reranking hurts performance on tight clusters

The gate computes the confidence spread of the top-5 vector results:

σ = max(scores[0:5]) - min(scores[0:5])

When σ is large, the vector model is confident — let it decide. When σ is tiny, it's a near-tie — reranker is gambling.

⛔ SKIP Reranker

σ > 0.08

Vector is confident.
Top-1 is clearly best.

✅ RUN Reranker

0.02 ≤ σ ≤ 0.08

Ambiguous ranking.
Let cross-encoder decide.

⛔ SKIP Reranker

σ < 0.02

Near-tie top-5.
Reranker gambling.

Implementation

export function computeRerankerGate(
  candidates: SearchResult[],
  mode: "off" | "hard" | "soft" = "off",
  threshold: number = 0.08,
  lowThreshold: number = 0.02,
): RerankerGateResult {
  const top5 = candidates.slice(0, 5);
  const scores = top5.map(c => c.score);
  const spread = Math.max(...scores) - Math.min(...scores);

  if (mode === "hard") {
    if (spread > threshold) return { shouldRerank: false, reason: "vector-confident" };
    if (spread < lowThreshold) return { shouldRerank: false, reason: "tied-set" };
    return { shouldRerank: true, reason: "ambiguous" };
  }
  // ... soft gate logic
}

Reciprocal Rank Fusion

Scale-invariant alternative to score blending

The problem: BM25 scores (5–20) and cosine similarities (0.2–0.6) are incompatible. Sigmoid normalization helped, but hardcoded midpoints caused saturation.

RRF(d) = Σ w_i / (k + rank_i)

Where k = 60 (standard smoothing constant). Documents appearing in both lists get double credit — an "agreement bonus."

TypeScript Implementation

export function blendByRank(
  pool: SearchResult[],
  rerankResults: Array<{ index: number; relevance_score: number }>,
  k: number = 60,
  vectorWeight: number = 1.0,
  rerankerWeight: number = 1.0,
): SearchResult[] {
  const rrfScores = new Map<number, number>();

  // Vector rank contribution
  for (let rank = 0; rank < pool.length; rank++) {
    rrfScores.set(rank, vectorWeight / (k + rank + 1));
  }

  // Reranker rank contribution (adds when doc in both lists)
  const sortedRerank = [...rerankResults].sort((a, b) => b.relevance_score - a.relevance_score);
  for (let rank = 0; rank < sortedRerank.length; rank++) {
    const idx = sortedRerank[rank].index;
    rrfScores.set(idx, (rrfScores.get(idx) ?? 0) + rerankerWeight / (k + rank + 1));
  }

  return [...rrfScores.entries()]
    .sort(([, a], [, b]) => b - a)
    .map(([idx, score]) => ({ ...pool[idx], score }));
}

Temporal Decay

Recent memories score higher

Exponential decay with a configurable floor:

decay(t) = floor + (1 - floor) × e^(-rate × ageDays)

Default: floor = 0.8, rate = 0.03. A 30-day-old memory scores 0.89× instead of 1.0×.

export function applyTemporalDecay(
  results: SearchResult[],
  opts?: { floor?: number; rate?: number }
): void {
  const floor = opts?.floor ?? 0.8;
  const rate = opts?.rate ?? 0.03;
  const now = Date.now();
  
  for (const r of results) {
    const rawTime = r.chunk.updated_at ? new Date(r.chunk.updated_at).getTime() : NaN;
    if (Number.isNaN(rawTime)) continue; // Guard against invalid timestamps
    
    const ageDays = Math.max(0, (now - rawTime) / (86400 * 1000));
    const decay = floor + (1 - floor) * Math.exp(-rate * ageDays);
    r.score *= decay;
  }
}

Maximal Marginal Relevance

Diversity filter prevents redundant results

MMR iteratively selects docs that are relevant but dissimilar to already-selected:

MMR = λ × Sim(q, d) - (1-λ) × max[Sim(d, d_j) for d_j in selected]

Default λ = 0.9 (relevance-biased). Uses cosine similarity on 4096-dim vectors.

export function mmrRerank(
  results: SearchResult[],
  lambda: number = 0.9,
  topK?: number
): SearchResult[] {
  if (results.length <= 1) return results;
  
  const selected: SearchResult[] = [];
  const remaining = [...results];
  
  while (remaining.length > 0 && (!topK || selected.length < topK)) {
    let bestIdx = 0;
    let bestScore = -Infinity;
    
    for (let i = 0; i < remaining.length; i++) {
      const relevance = remaining[i].score;
      let maxSim = 0;
      
      for (const s of selected) {
        const sim = cosineSimilarity(remaining[i].vector!, s.vector!);
        maxSim = Math.max(maxSim, sim);
      }
      
      const mmrScore = lambda * relevance - (1 - lambda) * maxSim;
      if (mmrScore > bestScore) {
        bestScore = mmrScore;
        bestIdx = i;
      }
    }
    
    selected.push(remaining.splice(bestIdx, 1)[0]);
  }
  
  return selected;
}
