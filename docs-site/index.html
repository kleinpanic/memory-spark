Dynamic Gating

Reranker Gate Logic

Skip the reranker when vector scores are already confident or tied. Save latency without losing accuracy.

Confidence Spread
σ = s₁ - sₖ
↓
Check σ
σ > 0.08
SKIP
Vector confident
0.02 ≤ σ ≤ 0.08
RUN
Ambiguous set
σ < 0.02
SKIP
Tied set

Hard Gate (Default)

Binary decision: skip or run. No middle ground. Simple and effective. 

if (σ > 0.08 || σ < 0.02) {
  skip_reranker();
} else {
  run_reranker();
}

Soft Gate

Dynamic weight adjustment based on σ. Higher σ = more trust in vector. 

vector_weight = min(1.0, σ / 0.08);
reranker_weight = 1.0 - vector_weight;

Scale-Invariant

Score Fusion & Diversity

RRF for hybrid merging, MMR for diversity, temporal decay for recency.

Reciprocal Rank Fusion

Merge vector and FTS results by rank, not score. Scale-invariant.

RRF(d) = Σ 1/(k + rᵢ)
k = 60 (default)

Maximal Marginal Relevance

Balance relevance vs. diversity. Avoid redundant results.

MMR = λ·sim(q,d) - (1-λ)·max sim(d,d')
λ = 0.9 (default)

Temporal Decay

Prioritize recent memories. Gentle exponential decay.

decay = floor + (1-floor)·e^(-rate·age)
floor=0.8, rate=0.03
