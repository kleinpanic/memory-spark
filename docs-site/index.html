Dynamic Reranker Gate

The key to 50% latency reduction without accuracy loss

High confidence spread — the top result is clearly better than the rest. Vector search already found the right answer.
SKIP RERANKER
σ < 0.02
Tied results — scores are too close to discriminate. Reranker won't add meaningful signal here.
SKIP RERANKER
0.02 ≤ σ ≤ 0.08
Ambiguous zone — results are neither clearly confident nor tied. Cross-encoder can help disambiguate.
RUN RERANKER
Reciprocal Rank Fusion
RRF(d) = Σ 1/(k + rank(d))
Scale-invariant merging of vector and BM25 results. Smoothing constant k=60 prevents rank #1 from dominating. Documents appearing high in both lists get an "agreement bonus."
